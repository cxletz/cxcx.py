Question: Complete the code to compute roots safely based on positive/negative and even/odd rules.
Answer:
def safe_root(a, b):
   if a >= 0:
       answer = a ** (1/b)
   else:
       if a % 2 == 0:
           answer = "Result is an imaginary number"
       else:
           answer = -(-a) ** (1/b)
   return answer
